# BuyGreen

BuyGreen is a mobile application that allows users to assess the ecological impact of a product simply by scanning its identification code before purchase. This tool helps consumers make informed and responsible decisions, encouraging companies to produce more sustainably and reducing harmful emissions into the environment.

## Features

- Scan product identification codes.
- Evaluate environmental impact based on parameters such as energy consumption, CO2 emissions, and percentage of recyclable material.
- Visual representation of environmental impact using a color-coded system (green for low impact, orange for medium impact, red for high impact).
- Multi-platform support through Flutter.
- Data management using MariaDB and communication via PHP APIs.

## Technologies Used

- [Flutter](https://flutter.dev/) - UI development framework.
- [MariaDB](https://mariadb.org/) - Relational database management system.
- [PHP](https://www.php.net/) - Scripting language used for APIs.
- [GitHub Actions](https://github.com/features/actions) - CI/CD for deployment and testing.
